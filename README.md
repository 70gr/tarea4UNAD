# Sistema Integral de Gestión de Clientes, Servicios y Reservas - FJ

## Descripción

Este proyecto consiste en el desarrollo de un sistema orientado a objetos que permite gestionar clientes, servicios y reservas para la empresa Software FJ.

El sistema está diseñado bajo principios de programación orientada a objetos, garantizando estabilidad, robustez y manejo adecuado de errores sin el uso de bases de datos.

---

## Objetivos

- Implementar clases abstractas y herencia
- Aplicar encapsulación y polimorfismo
- Manejar excepciones de forma robusta
- Garantizar la continuidad del sistema ante errores

---

## Estructura del Proyecto

- `cliente.py` → Gestión de clientes  
- `servicio.py` → Servicios y clases derivadas  
- `reserva.py` → Gestión de reservas  
- `excepciones.py` → Manejo de errores personalizados  
- `main.py` → Ejecución del sistema  
- `logs.txt` → Registro de errores  

---

## Tecnologías utilizadas
- Python
- Programación Orientada a Objetos (POO)
